Loja Online Cantinho da Prof.ª Ana
Aqui está um guia completo passo a passo de como o seu site funciona, dividido entre a visão do Cliente e a visão do Administrador (Dono da Loja).
🏪 Visão Geral
Este é um sistema completo de Loja de Conteúdos Digitais (e-books, apostilas, cursos). A grande diferença deste projeto é que não existe frete nem valor mínimo de compra, pois tudo é entregue digitalmente ("Acesso Imediato"). Além disso, ele possui um Painel Administrativo poderoso onde você pode mudar textos, cores e produtos sem precisar mexer no código.
👤 1. Para o Cliente (A Loja)
O cliente navega pela loja para comprar materiais. Veja o fluxo:
Navegação e Busca:
No topo, o cliente vê as categorias (Matemática, Português, etc.).
Existe uma barra de busca para encontrar materiais específicos.
Visualização dos Produtos:
Os produtos aparecem em grade.
Cada produto tem um selo verde escrito "Acesso Imediato", indicando que é um produto digital (download ou acesso online após a compra).
O cliente pode clicar em "Adicionar" ou clicar na imagem para ver detalhes.
Carrinho de Compras:
Ao clicar no ícone do carrinho, abre-se uma barra lateral mostrando os itens escolhidos.
O cliente pode alterar quantidades ou remover itens.
Nota: Não há cálculo de frete, apenas o total dos produtos.
Checkout (Pagamento):
Ao clicar em "Finalizar Compra", o cliente escolhe a forma de pagamento.
PIX: Gera um QR Code (simulado) e um código "copia e cola".
Cartão: Formulário para dados do cartão (simulação).
Boleto: Opção de gerar boleto.
Chatbot "Edu":
No canto inferior esquerdo, há um ícone de um avatar. Ao clicar, abre-se um chat com a assistente virtual "Edu", que responde dúvidas frequentes sobre a loja.

Aqui é onde você controla tudo. Para acessar, clique em "Painel Admin" no topo do site.

Dentro do painel, você tem as seguintes seções:
A. Dashboard (Painel Inicial)
Mostra resumos de vendas, pedidos e número de produtos cadastrados.
Gráfico de vendas da semana.
B. Conteúdo do Site (✨ Novidade Importante)
Aqui você pode editar textos da loja sem saber programação:
Top Bar: Mude o texto de boas-vindas.
Features: Edite os ícones e textos dos benefícios (ex: mudar "Acesso Imediato" para "Download Rápido").
Footer: Altere o rodapé, links e direitos autorais.
Páginas Info: Edite o texto das páginas "Sobre Nós", "Como Funciona", "Políticas", etc.
Títulos: Mude os títulos das seções da home.
C. Produtos
Adicionar Novo: Crie novos produtos definindo nome, categoria, preço, imagem e descrição.
Editar/Excluir: Altere preços ou remova produtos existentes.
Nota: Ao criar um produto digital, o sistema já configura automaticamente o tipo de entrega como "Acesso Imediato".
D. Categorias
Crie novas categorias de materiais ou edite as existentes (ícones e cores).
E. Banners (Slider)
Altere os banners que ficam girando no topo da loja (textos e imagens).
F. Cupons
Crie cupons de desconto (ex: PROMO10 para 10% OFF) para usar no checkout.
G. Avatar e Bot
Nome do Bot: Mude o nome da assistente virtual (de "Edu" para o que quiser).
Avatar: Troque a foto do bot fazendo upload de uma nova imagem ou colando uma URL.
H. Temas (Cores)
Temas Prontos: Botões para mudar a cor da loja inteira com um clique (Rosa, Azul, Verde, Dark Mode, etc.).
Personalizado: Seletor de cores para definir a cor exata da sua marca.
💡 Detalhes Técnicos Importantes
Armazenamento Local (LocalStorage):
Este site salva todas as suas edições (mudanças de cor, novos produtos, textos alterados) na memória do seu navegador.
Vantagem: Você pode fechar o navegador e abrir de novo que as alterações estarão lá.
Cuidado: Se você limpar o "cache/histórico" do navegador, os dados voltam ao padrão original. Para um site profissional na internet, seria necessário um banco de dados real.
Responsividade:
O site se adapta perfeitamente a celulares, tablets e computadores.
Pagamento Simulado:
O checkout simula uma compra real para fins de demonstração. O QR Code do PIX é gerado aleatoriamente apenas para visualização.
